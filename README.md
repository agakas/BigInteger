# BigInteger
Реализация класса BigInteger для работы с длинными целыми числами

---

## Постановка задачи

Должны поддерживаться следующие операции

- Сложение, вычитание, умножение, деление, остаток по модулю, работающие так же, как и для int: составное присваивание с этими операциями. Умножение, деление и остаток по модулю должны работать не дольше, чем за О(п^2)
- Унарный плюс, унарный минус, префиксные и постфиксные инкремент и декремент. Префиксный инкремент и декремент должны работать за O(1) в среднем, а также не делать избыточных аллокаций
- Оператор <=>, сравнения должны работать с его помощью, а самих операторов ≤, <=, ≥, >= быть не должно
- Оператор ==
- Вывод в поток (должен корректно обрабатывать и числа с ведущими нулями) и ввод из потока
- Метод ToString(), возвращающий строковое представление числа
- Конструирование из int64_t. uint64 t (в том числе неявное преобразование, когда это надо), std: : string (из любой строки из цифр, в частности, допустимы ведущие нули)
- Преобразование в bool (должно работать в условных выражениях)
- Литеральный суффикс _bі для написания литералов типа BigInteger. Он должен работать как от чисел (вида 37183123127313234632762740326741_bi), так и от C-style строк (вида
2346327624203647324"_bi)
